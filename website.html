💡 Pro Tips

  • Always run sona status first to diagnose issues
  • Check log files at ~/.sona/sona.log for detailed error information
  • Ensure you have sufficient disk space for large audio/video files
  • Use the interactive mode sona interactive for guided troubleshooting
