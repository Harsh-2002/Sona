How to Use

1

Get API Key

Sign up at assemblyai.com to get your free API key.

2

Configure Sona

Set your API key in the terminal.

sona config set api_key "your_api_key_here"
3

Install Dependencies

Install yt-dlp and FFmpeg automatically.

sona install
4

Verify Installation

Run a status check to ensure everything is ready.

sona status

Transcribe a Local File

sona transcribe audio.mp3

Transcribe a YouTube Video

sona transcribe "https://youtube.com/watch?v=VIDEO_ID"

Specify an Output Path

sona transcribe audio.mp3 --output transcript.txt

Use a Different AI Model

sona transcribe audio.mp3 --model best

Use Interactive Mode

sona interactive
